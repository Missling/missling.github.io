Project #1: This website

For this week project, I created my very own website from scratch without relying on a front-end framework such as Bootstrap. Prior to this project, my only exposure to HTML/CSS was completing the course on Codecademy. Because Codecademy taught HTML4, I had to read up on the new semantic elements that were introduced in HTML5.

To begin this project, I learned about the differences between an UX Designer and UI Designer, which taught me how complicated the design process actually is. From there, I wireframed the layout of my future website and with a little HTML/CSS magic, I turned my wireframe into a living, working website. During this process, my biggest struggle was working with CSS positioning. Luckily, I was introduced to Chrome DevTools, which became an invaluable tools for myself to edit elements and help me with my toughest challenge, CSS positioning. Although most of my website was created by myself, the template for my blog was actually the result of a pair-programming session. I had a great partner, which lead to a a fantastic pairing experience. I can now see why Dev Bootcamp place such a strong emphasis on pair-programming. My website is far from complete, but I plan on growing my website as I continue to learn more about HTML/CSS and programming in general.
