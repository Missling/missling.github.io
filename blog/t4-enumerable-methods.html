Map Method

January 17, 2015

As an object-oriented programming language, almost everything in Ruby is considered an object. With each object, there are built-in abilities called methods that tell the object what to do. For this post, I will be discussing the enumerable method, map. You maybe wondering what is an enumerable? Well, to give a simple answer, enumerable provides a way to sort and search through elements of the collection classes.

The map method takes an array or a range, and passes each element of the array or range through a block. From there, the new values returned from the block are output in a new array. The following examples demonstrate the map methods.

Example 1: 

        array = [1, 2, 3, 4, 5]

        array.map {|number| number + 10}

        output a new array: [11, 12, 13, 14, 15]

Example 2: 

        range = 1..5

        range.map {|number| number + 10}

        output: [11, 12, 13, 14, 15]
